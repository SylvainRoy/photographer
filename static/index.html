About this tool

Worried about privacy? Only the coordinates of the points — on the picture (x, y) and on the map (latitude, longitude) — are sent to the server. The picture itself never leaves your browser.

This is an open-source toy project. The server that computes the location exposes a JSON API you can call directly. This tool comes with no guarantee.
